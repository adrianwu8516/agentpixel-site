新時代文書辦公

Agent 產出的是 Markdown、HTML，不是 .docx。當文件本身就是 Agent 的工作成果，你需要能立刻看見、立刻改、立刻找到的環境——而不是把它匯出到另一套辦公軟體。

雙開與即時編輯

分屏雙開
Terminal 與文件並排

  • 分屏雙開：Terminal 與文件同時顯示，直觀的並排工作流。
  • 即時編輯：HTML / Markdown 檔案當場編輯，體驗貼近主流編輯器。
  • Agent 驅動：以 MD / HTML 為主要產出，取代 Word、PowerPoint 舊式流程。
找檔案不再靠猜

即時文件瀏覽器

  • 解決辦公痛點：Agent 產出的檔案到底存在哪？
  • Terminal 操作到哪，File Explorer 就同步顯示到哪。
  • 隨時定位、開啟、瀏覽 Agent 建立的每一份檔案。
帳單心裡有數

訂閱 Token 消耗追蹤

  • 管理面板內建即時 Token 用量看板。
  • 工作中隨時掌握帳號額度消耗，不必事後才發現超用。
  • 同時支援 Claude 與 Codex 雙軌追蹤。

專為辦公優化的 Terminal

同時開好幾個 Agent，是效能問題，也是狀態管理問題。這一層的功能，是讓 Terminal 本身撐得住多 Agent 的日常辦公強度。

輕量、不失控

資源管理器
＋ 狀態持久化

  • 多 Agent 同時運作最容易吃滿系統資源，內建即時資源監控面板。
  • 狀態持久化：關閉重開，Tab 佈局與工作現場完整還原。
離開電腦，工作不停

跨端聯動
與遠端監控

  • 電腦端一鍵開啟 Remote，Terminal 立即聯動到手機。
  • 不必守在電腦前，隨時隨地掌握 Agent 工作狀態。
一個 Manager，管全部 Tab

高階多標籤管控實驗性

  • Manager 模式：由一個主 Manager 統籌控制其他所有 Tab。
  • 更高層級的全局管理，目前仍在實驗階段。
